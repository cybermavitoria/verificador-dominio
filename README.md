# Verificador de Domínio

Ferramenta em Python que verifica o IP de um domínio e informa se foi encontrado com sucesso.

## Como usar

1. Rode o script:
python verificador.py

2. Digite o domínio que deseja verificar (ex: google.com)

## O que o script verifica

- IP do domínio
- Data e hora da consulta
